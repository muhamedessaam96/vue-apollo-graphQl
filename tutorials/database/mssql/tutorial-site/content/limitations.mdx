## Deprecated database engine features

MS SQL Server 2019 does not deprecate any features beyond those deprecated in the earlier releases.
MS SQL Server objects and counters are used by the System Monitor to monitor the performance of SQL Server instances.

The dynamic view `sys.dm_os_performance_counters` (Transact-SQL) provides all counters generated by SQL Server.

You can execute the following query to view the deprecated counters.

```SQL
SELECT * FROM SYS.dm_os_performance_counters
WHERE object_name = 'SQLServer:Deprecated Features';
```
